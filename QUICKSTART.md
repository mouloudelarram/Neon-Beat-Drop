# 🚀 Quick Start Guide - Magic Piano Quest

## 5-Minute Setup

### 1. Install Dependencies
```bash
cd Neon-Beat-Drop
flutter pub get
```

### 2. Run the Game
```bash
flutter run
```

### 3. Build Release APK
```bash
flutter build apk --release
```

---

## 🎮 How to Play

1. **Tap falling tiles** in the bottom 150px of the screen
2. **Build combos** by tapping consecutive tiles
3. **Don't miss** - each miss costs one life (3 total)
4. **Watch the score** multiply with higher combos
5. **Beat your high score!**

---

## 🎨 Game Features

| Feature | Details |
|---------|---------|
| 🎯 Lanes | 4 colored columns (Magenta, Cyan, Yellow, Purple) |
| ⭐ Scoring | 10 pts × combo multiplier |
| 💔 Lives | 3 hearts per game |
| 🚀 Difficulty | Speed increases 5% per 5 tiles |
| ✨ Particles | Explosion effects on successful taps |
| 🎵 Audio | Tap sounds with graceful fallback |
| 🎨 Theme | Neon/vibrant for kids |
